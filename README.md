# Тренажёр реакции Go/No-Go

Учебное веб-приложение на React + TypeScript для проверки скорости реакции и внимательности пользователя.

## Что реализовано

- стартовый экран с правилами;
- тестовая сессия из 20 попыток;
- случайная задержка перед сигналом;
- случайный выбор сигнала Go или No-Go;
- обработка нажатия клавиши Space;
- подсчёт успешных реакций, пропусков, ложных и ранних нажатий;
- экран результатов;
- история прохождений через localStorage;
- тесты для расчёта статистики.

## Запуск проекта

```bash
npm install
npm run dev
```

После запуска открыть ссылку из терминала, обычно:

```text
http://localhost:5173/
```

## Сборка

```bash
npm run build
```

## Тесты
